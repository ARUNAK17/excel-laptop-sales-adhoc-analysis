# Laptop Sales Ad-Hoc Analysis (Excel)

## Overview
This project is an **ad-hoc Excel analysis** conducted to explore **laptop sales performance across regions** and evaluate the **impact of promotion codes on total sales value**.

The analysis focuses on quick, business-driven insights rather than automated reporting or dashboarding.

---

## Business Question
- How do laptop sales vary by region?
- What is the impact of **promotion codes** on total laptop sales?
- Do promotions significantly increase revenue across regions?

---

## Data Description
The dataset contains transactional sales records with the following attributes:
- Date, Region, Product
- Quantity, Unit Price, Discount
- Promotion Code
- Total Price
- Store and Customer details

The analysis focuses specifically on **Laptop** sales.

---

## Analysis Performed
- Data cleaning and filtering using **Excel Power Query**
- Ad-hoc aggregation of laptop sales by:
  - Region
  - Promotion Code
- Comparison of:
  - **Sales with promotion codes**
  - **Sales without promotion codes**
- Region-wise revenue comparison to identify promotion effectiveness

---

## Key Insights
- Promotion codes contribute significantly to higher total sales value
- The impact of promotions varies by region
- Certain regions show stronger uplift from promotional campaigns than others
- Ad-hoc analysis helps quickly validate business assumptions without complex tooling

---

## Tools Used
- Microsoft Excel
- Power Query
- Pivot-style aggregation
- Manual exploratory analysis
